# Transcripts Index

Exemplars of the three master teaching styles the system emulates in narrative-Socratic teaching (Principle 16). The transcript files are **shortened to their best verbatim exemplar passages, organized by teaching beat** — they are calibration for *texture and pacing*, not rules (the operational rules are in the Narrative-Socratic report and skill §16). **Do not read at startup.** Before the first narrative (Path A) session, read this index; pull the relevant transcript on demand when you want a worked example of a specific beat.

| Teacher (file) | Use for | Signature move |
|---|---|---|
| `Floathead-Physics-Transcripts.md` | the **default** method — any concept taught through a historical character | Interrogates a named historical physicist (Schrödinger, Feynman) through reported dialogue until the model itself forces the next uncomfortable question |
| `Veritasium-Transcripts.md` | **dislodging a misconception** | Validates the wrong belief → dramatizes the exact historical moment it collapsed → only then delivers the replacement, usually leaving one thread visibly unresolved |
| `3Blue1Brown-Transcripts.md` | **math / structural** concepts | Organizes the whole session around one driving question; introduces each object only when the question demands it; geometric intuition before algebra |

### Which transcript best illustrates each beat
- **Problem / exposition setup** → 3Blue1Brown (deferred driving questions stated up front), Floathead (the planetary-model instability), Veritasium (the drifting-rock cold open).
- **Analogy anchor** (loading a source domain first) → Veritasium (mirror symmetry on a triangle before it's used as a tool); 3Blue1Brown (Coulomb's law introduced by the question it answers).
- **Structural mapping** → 3Blue1Brown (cosine-shadow formula from the right-triangle slice); Floathead ("kid in the park" → probability cloud).
- **Analogy breakdown** ("the breakdown IS the concept") → 3Blue1Brown (strip away "does shape matter? no; does linearity? yes" → generalizes to all convex solids); Floathead (slicing the sphere to generate p/d orbitals).
- **Misconception dramatization** → Veritasium (Einstein's pseudotensor "fix" demolished by Noether).
- **Socratic question through narrated character voice** → Floathead (the element-by-element walk: "Feynman says, Mahesh, what do you think happens if you heat a hollow metal cube with a hole?").
- **Productive confusion + cliffhanger** → Veritasium ("They don't exist." / "I wish they did… it's not a proof" — oldest unsolved problem left open).
- **Resolution with consequence** → Veritasium (Nakamura's two-flow reactor → the LED revolution); Floathead (discrete energy levels → the hydrogen spectrum via e=hf).
- **Deeper question** (answer opens a harder problem) → Floathead ("the octet rule…"), 3Blue1Brown ("which mindset correlates with discovering new math? …a blend").
