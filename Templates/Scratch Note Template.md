# Scratch Note Template

The working document for a learning session. Created in `00 - Inbox/` at the start of a
new topic, updated briefly during the session, then moved to `.note-information/Scratch/`
once a permanent note is created. Minimal — context for future sessions, not a transcript.

```
---
date: [TODAY]
last-modified: [TODAY]
type: scratch
status: active
cognitive-state: generative
tags: [learning-session, scratch]
topic: [TOPIC]
---
## [TOPIC] — Session [DATE]
### Pretest — What I Already Believe
### Attempt 1
### Attempt 2 (if reached — even if wrong, explain why you think it's wrong)
### Questions That Came Up
### Sources to Check
### Key Concepts Taught (2–4 sentences max)
### After Consolidation — What I Got Right / Wrong
### Status   [e.g. "Permanent note created: [[Title]]" or "Lesson in progress"]
```
